# Tutor Mart frontend phase 1

## What is included
- public home with Google sign-in button
- onboarding page for initial role choice
- guardian dashboard
- tutor dashboard
- switch/add role actions
- protected role pages
- direct conversation page with realtime updates
- text + audio messaging
- edit/delete/reply UI

## Local run
1. Copy `.env.example` to `.env.local`
2. `npm install`
3. `npm run dev`
